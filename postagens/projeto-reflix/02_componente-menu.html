Etapa 02 - Criar componentes

Estrutura básica do componente no projeto

Criar pastas e arquivos referentes aos componentes

  • dentro da pasta src
    • criar a pasta components
  • dentro da pasta components
    • criar a pasta Menu
  • dentro da pasta Menu criar os arquivos
    • index.js
    • Menu.css
> src
	> components
		> Menu
			- index.js
			- Menu.css

Se for desenvolver o projeto com styled-components utilizar estrutura específica, apenas com o index.js na pasta do componente, que conterá tanto o HTML quanto o CSS.

Estrutura básica do index.js do componente

No arquivo Menu/index.js

  • importar a biblioteca React
  • importar o arquivo CSS do componente
  • criar a função do componente
  • inlcuir o return da função
  • exportar a função
import  React  from  "react";
import  './Menu.css';

function  Menu() {
	return(
	
	);
}
export  default  Menu;

Essa é a estrutura básica de componentes React. Toda a definição HTML do componente será declarada dentro do return da função.

Estrutura HTML do componente Menu

No arquivo Menu/index.js

  • dentro do return da função Menu
    • declarar a tag nav com a classe menu
  • a tag nav conterá duas tags de link a
    • do logo
      • com direcionamento para a “home” (/)
      • contendo a tag de imagem img com os atributos src e alt
    • do botão “Novo vídeo”
      • com a classe botao-menu
      • com direcionamento provisório para “home” (/)
      • contendo o texto “Novo vídeo”
return (
	<nav  className="menu">
		<a  href="/">
			<img  src=""  alt=""  />
		</a>
		<a  className="botao-menu"  href="/">Novo vídeo</a>
	</nav>
);

Incluir pasta de imagens no projeto

  • dentro da pasta src
    • criar a pasta assets
  • dentro de assets
    • criar a pasta img
> src
	> assets
		> img

A pasta assets guardará todos os recursos adicionais do projeto, como imagens, fontes e outros.

Criar Logotipo

  • site “Font Meme” ( https://fontmeme.com/pt/fonte-da-netflix/ )
  • gerar imagem com:
    • tamanho 120px
    • cor #2A7AE4
  • exportar e salvar imagem na pasta de imagens
    • src/assets/img/devflix-logo.png

Importar imagem do logo

No arquivo Menu/index.js

  • abaixo da linha de importação do Menu.css
  • importar a propriedade Logo indicando o caminho do arquivo
  • na tag img definir os valores dos atributos:
    • className com o seletor logo
    • src com a propriedade Logo
    • alt com o texto explicativo da imagem
import  Logo  from  '../../assets/img/devflix-logo.png';
...
return (
	<nav  className="menu">
		<a  href="/">
			<img  className="logo"  src={Logo}  alt="Logotipo Devflix"  />
		</a>
		<a  className="botao-menu"  href="/">Novo vídeo</a>
	</nav>
);

Incluir componente na aplicação

No arquivo src/App.js

  • dentro do return da função “App”
    • incluir a tag <Menu />

O VSCode faz a importação automática do componente ao declarar a tag Menu. Caso esta ação automática não ocorra, a importação deverá ser feita manualmente, digitando no início do arquivo, abaixo da linha de importação do React:

 import  Menu  from  './components/Menu';

Código do arquivo src/App.js

import  React  from  'react';
import  Menu  from  './components/Menu';

function  App() {
	return (
		<div>
			<Menu  />
			<h1>Hello, World!!</h1>
		</div>
	);
}

export  default  App;
